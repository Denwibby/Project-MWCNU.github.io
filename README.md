# NU School Website

## Overview
The NU School Website is designed to provide information about the Majelis Wakil Cabang Nahdlatul Ulama's educational institutions, including TK (Taman Kanak-Kanak), SD Maarif, MTs (Madrasah Tsanawiyah), and SMK (Sekolah Menengah Kejuruan). This website serves as a comprehensive resource for parents, students, and the community, showcasing the school's mission, vision, and various educational programs.

## Installation
To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd nu-school-website
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Open `src/index.html` in your web browser to view the website.

## Usage
The website includes the following sections:

- **Profile**: A general overview of the school, including its mission and vision.
- **TK**: Information about the Taman Kanak-Kanak section, including curriculum and activities.
- **SD Maarif**: Details about the SD Maarif section, including educational programs and achievements.
- **MTs**: Information about the Madrasah Tsanawiyah section, highlighting academic offerings and extracurricular activities.
- **SMK**: Information about the Sekolah Menengah Kejuruan section, focusing on vocational training and career preparation.
